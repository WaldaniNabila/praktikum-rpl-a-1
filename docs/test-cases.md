# Dokumen Test Case - JobHub

Dokumen ini berisi daftar skenario pengujian (Test Cases) menyeluruh untuk platform portal lowongan kerja **JobHub**. Pengujian ini dirancang untuk memastikan kualitas aplikasi, mencakup berbagai fitur utama seperti registrasi pengguna (Pelamar & Perusahaan), alur melamar pekerjaan, manajemen lowongan, hingga fungsi moderasi oleh Admin. 

## Ringkasan Eksekusi
- **Total Test Cases:** 18 skenario
- **Pass (Lulus):** 17 skenario
- **Fail (Gagal):** 1 skenario (Catatan: Bug validasi format email longgar pada TC-002)
- **Tingkat Kelulusan:** 94.4%
