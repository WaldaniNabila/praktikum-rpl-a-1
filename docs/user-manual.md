# User Manual JOBHUB

## 1. Pendahuluan

### Tentang JOBHUB
JOBHUB adalah platform rekrutmen online yang dirancang untuk mempertemukan pencari kerja (Job Seeker) dengan berbagai perusahaan yang sedang membuka lowongan pekerjaan. Platform ini menyediakan kemudahan dalam mencari pekerjaan, melamar secara langsung, serta memfasilitasi perusahaan dalam menemukan kandidat terbaik.

### Tujuan Dokumen
Dokumen User Manual ini disusun dengan tujuan memberikan panduan langkah demi langkah kepada pengguna (Pelamar, Perusahaan, dan Admin) tentang cara menggunakan berbagai fitur yang tersedia di aplikasi JOBHUB secara efektif.

### Persyaratan Sistem
Untuk dapat menggunakan JOBHUB dengan optimal, berikut adalah persyaratan sistem yang disarankan:
- **Browser:** Versi terbaru dari Google Chrome, Mozilla Firefox, Safari, atau Microsoft Edge.
- **Koneksi Internet:** Diperlukan koneksi internet yang stabil untuk mengakses website, mengunggah file (seperti CV atau foto profil), dan memuat data secara real-time.

## 2. Memulai

### Cara Mengakses Website
1. Buka aplikasi web browser di perangkat Anda (Komputer/Laptop atau Smartphone).
2. Ketikkan URL atau alamat website JOBHUB di address bar dan tekan **Enter**.
3. Anda akan diarahkan ke halaman utama (Landing Page) JOBHUB.

### Cara Registrasi (Pelamar & Perusahaan)
Bagi pengguna baru, ikuti langkah-langkah berikut untuk membuat akun:
1. Klik tombol **Daftar** atau **Register** yang terdapat di pojok kanan atas halaman.
2. Masukkan nama lengkap, alamat email yang valid, dan buat kata sandi (password).
3. **Pilih Peran:**
   - Pilih **Pelamar** jika Anda ingin melamar pekerjaan.
   - Pilih **Perusahaan** jika Anda mewakili perusahaan yang ingin memasang lowongan.
4. Klik tombol **Daftar** untuk menyelesaikan proses registrasi.

### Cara Login & Logout
- **Login:**
  1. Klik tombol **Login** di pojok kanan atas halaman.
  2. Masukkan email dan password yang telah Anda daftarkan.
  3. Klik tombol **Masuk**.
- **Logout:**
  1. Klik pada icon atau nama profil Anda di pojok kanan atas.
  2. Pilih menu **Logout** atau **Keluar** untuk mengakhiri sesi.

### Cara Reset Password
1. Pada halaman Login, klik tautan **Lupa Password?**.
2. Masukkan alamat email yang terdaftar di akun JOBHUB Anda.
3. Klik tombol **Kirim Link Reset**.
4. Cek kotak masuk (inbox) email Anda untuk menemukan link reset password. Klik link tersebut dan ikuti instruksi untuk membuat password baru.

## 3. Panduan Job Seeker (Pelamar)

### Mencari Lowongan (Search & Filter)
1. Setelah login, akses menu **Cari Lowongan**.
2. Gunakan kolom pencarian untuk mengetikkan posisi pekerjaan, nama perusahaan, atau kata kunci lainnya.
3. Gunakan fitur **Filter** untuk menyaring lowongan berdasarkan kategori, lokasi, atau rentang gaji yang diinginkan.

### Melihat Detail Lowongan
1. Dari hasil pencarian, klik pada salah satu judul lowongan pekerjaan.
2. Anda akan melihat informasi lengkap seperti deskripsi pekerjaan, kualifikasi yang dibutuhkan, benefit, gaji, serta profil singkat perusahaan.

### Melamar Pekerjaan (Apply + Upload CV)
1. Pada halaman detail lowongan, klik tombol **Lamar Pekerjaan**.
2. Anda akan diminta untuk melengkapi formulir lamaran (jika ada) dan **Mengunggah CV** (dalam format PDF, ukuran maksimal biasanya 2MB atau sesuai ketentuan).
3. Setelah memastikan semua data benar, klik tombol **Kirim Lamaran**.

### Memantau Status Lamaran
1. Buka menu **Lamaran Saya** di dashboard Pelamar.
2. Anda dapat melihat daftar semua lowongan yang telah Anda lamar beserta **Status** terkininya (seperti: Menunggu, Diproses, Diterima, atau Ditolak).

### Menyimpan Lowongan (Bookmark)
1. Jika Anda menemukan lowongan menarik namun belum ingin langsung melamar, klik ikon **Simpan** atau **Bookmark** pada lowongan tersebut.
2. Daftar lowongan yang disimpan dapat dilihat kapan saja melalui menu **Lowongan Tersimpan**.

### Mengedit Profil & Upload Foto
1. Buka menu **Profil Saya**.
2. Anda dapat memperbarui informasi pribadi, latar belakang pendidikan, dan pengalaman kerja.
3. Untuk mengubah foto profil, klik tombol **Ubah Foto** dan pilih file foto terbaru dari perangkat Anda, lalu klik **Simpan Profil**.

## 4. Panduan Perusahaan

### Melengkapi Profil Perusahaan
1. Setelah login sebagai Perusahaan, masuk ke menu **Profil Perusahaan**.
2. Lengkapi informasi penting seperti nama perusahaan, deskripsi singkat, industri, lokasi kantor, dan unggah logo perusahaan.
3. Klik **Simpan** agar profil perusahaan terlihat menarik bagi para pelamar.

### Memposting Lowongan Baru
1. Buka menu **Kelola Lowongan** dan klik tombol **Tambah Lowongan Baru**.
2. Isi formulir pembuatan lowongan secara detail (Judul pekerjaan, kategori, deskripsi tugas, persyaratan, gaji, dan batas waktu lamaran).
3. Klik **Posting Lowongan**. (Catatan: Lowongan mungkin memerlukan persetujuan Admin sebelum tayang di halaman publik).

### Mengedit & Menghapus Lowongan
- **Edit:** Pada menu **Kelola Lowongan**, pilih lowongan yang ingin diubah, klik tombol **Edit**, lakukan perubahan yang diperlukan, lalu klik **Simpan**.
- **Hapus:** Jika lowongan sudah tidak relevan atau posisi sudah terisi, klik tombol **Hapus** pada lowongan tersebut untuk menghapusnya dari sistem.

### Melihat Daftar Pelamar
1. Pada daftar lowongan Anda, klik opsi **Lihat Pelamar** pada lowongan tertentu.
2. Anda akan melihat daftar semua kandidat yang telah mengirimkan lamaran untuk posisi tersebut, lengkap dengan tautan untuk mengunduh CV dan melihat profil mereka.

### Mengubah Status Lamaran
1. Di halaman daftar pelamar suatu lowongan, klik pada salah satu kandidat.
2. Anda dapat mengevaluasi pelamar dan mengubah status lamarannya (misalnya dari "Menunggu" menjadi "Diproses", "Diterima", atau "Ditolak") menggunakan menu dropdown atau tombol aksi yang tersedia.

## 5. Panduan Admin

### Approve & Reject Lowongan
1. Login sebagai Admin dan buka menu **Moderasi Lowongan**.
2. Admin dapat melihat daftar lowongan baru yang diposting oleh perusahaan.
3. Klik **Approve** agar lowongan dapat dilihat publik, atau **Reject** jika lowongan melanggar kebijakan platform.

### Verifikasi Perusahaan
1. Buka menu **Kelola Perusahaan**.
2. Admin bertanggung jawab meninjau data perusahaan yang baru mendaftar.
3. Klik tombol **Verifikasi** untuk memberikan status valid kepada perusahaan, sehingga pelamar lebih percaya terhadap perusahaan tersebut.

### Kelola Pengguna
1. Buka menu **Manajemen Pengguna**.
2. Admin dapat melihat, mencari, dan mengelola semua akun yang ada di sistem (Pelamar dan Perusahaan).
3. Admin memiliki wewenang untuk membekukan (suspend) atau menghapus akun pengguna yang melanggar aturan.

### Kelola Kategori
1. Buka menu **Kelola Kategori**.
2. Admin dapat menambahkan kategori pekerjaan baru, mengedit nama kategori, atau menghapus kategori yang tidak diperlukan agar fitur pencarian lebih terstruktur.

## 6. Troubleshooting

### Tidak Bisa Login
- Pastikan email dan password yang Anda masukkan sudah benar.
- Perhatikan penggunaan huruf besar/kecil pada password (Caps Lock).
- Jika Anda masih gagal login, coba gunakan fitur **Lupa Password** untuk mereset kata sandi.

### Lupa Password
- Klik tautan **Lupa Password?** pada halaman Login.
- Masukkan email terdaftar dan ikuti instruksi yang dikirimkan ke email Anda untuk membuat password baru.
- Jika email reset tidak muncul di kotak masuk utama, periksa folder Spam atau Junk di email Anda.

### Gagal Upload File
- **Ukuran File Terlalu Besar:** Pastikan ukuran file CV atau foto yang Anda unggah tidak melebihi batas maksimal (biasanya 2MB atau 5MB sesuai ketentuan platform).
- **Format File Tidak Sesuai:** Periksa kembali ekstensi file Anda. CV umumnya harus berformat `.pdf`, sedangkan foto harus berformat `.jpg`, `.jpeg`, atau `.png`.
- **Koneksi Internet:** Pastikan koneksi internet Anda stabil saat melakukan proses unggah file.
